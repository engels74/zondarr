# Zondarr Product Overview

Zondarr is a unified invitation and user management system for media servers (Plex, Jellyfin).

## Core Purpose

Enable administrators to create invitation codes that grant users access to media libraries with:
- Configurable permissions
- Library restrictions
- Expiration policies

## Key Features

- Multi-platform support: Plex, Jellyfin (extensible to Emby, Audiobookshelf)
- Invitation code generation and redemption
- User synchronization across media servers
- OAuth integration (Plex OAuth)
- Admin dashboard for managing users, servers, and invitations

## Target Users

- Media server administrators managing shared libraries
- Users receiving invitations to access media content
